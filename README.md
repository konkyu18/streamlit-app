## データ使用上の注意点
本データには「在学者」と「卒業者」の両方が含まれているが、本アプリでは学生のアルバイト状況について着目するため、「在学者」だけに限定した。
また本アプリでは全47都道府県のデータではなく、**東京都・長野県・静岡県**の3都県にデータを限定して使用している。
これは、全都道府県を一度に表示するとグラフが細かくなりすぎて視認性が下がる点と、データ処理による動作により重くなることを防ぐためです。

 この3つを選んだ理由
* **東京都:** 都会の代表例として、地方との比較を行うため。
* **静岡県:** 私の出身地であり、土地勘があるため。
* **長野県:** 現在住んでいる地域であり、身近なデータとして分析したいため。

## 使用した「授業で扱っていないUI」
本アプリでは、ユーザビリティ向上のために以下の5つの未使用UIを使いました。

1.  **st.set_page_config**
    * **目的:** ブラウザタブのタイトルを設定し、Webアプリとしての体裁を整えるため。
    * **使用箇所:** アプリ起動時の初期設定。
2.  **st.metric**
    *  **目的:** 「学生数」や「有業率」などの最重要指標を、視認性の高い大きな文字で強調表示するため。
    * **使用箇所:** メイン画面上部のハイライトエリア。
3.  **st.tabs**
    * **目的:** 「学校種別比較」「地域比較」「データの詳細とダウンロード」という異なる情報を整理し、スクロールせずに切り替えて閲覧できるようにするため。
    * **使用箇所:** メイン画面中央のコンテンツ切り替え部分。
4.  **st.expander**
    * **目的:** 詳細なデータテーブルは常時表示すると画面がになるため、必要なユーザーのみが展開して見られるようにするため。
    * **使用箇所:** 「データの詳細とダウンロード」タブ内。
5.  **st.download_button**
    * **目的:** ユーザーが分析結果の元データを見たり、利用できるようにするため
    * **使用箇所:** 「データの詳細とダウンロード」タブ内。
