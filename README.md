## データ使用上の注意点
本データには「在学者」と「卒業者」の両方が含まれているが、本アプリでは学生のアルバイト状況について着目するため、「在学者」だけに限定した。　　
また本アプリでは全47都道府県のデータではなく、**東京都・長野県・静岡県**の3都県にデータを限定して使用している。
これは、全都道府県を一度に表示するとグラフが細かくなりすぎて視認性が下がる点と、データ処理による動作により重くなることを防ぐためです。
またこの3つを選んだ理由は
* **東京都:** 都会の代表例として、地方との比較を行うため。
* **静岡県:** 私の出身地であり、土地勘があるため。
* **長野県:** 現在住んでいる地域であり、身近なデータとして分析したいため。
